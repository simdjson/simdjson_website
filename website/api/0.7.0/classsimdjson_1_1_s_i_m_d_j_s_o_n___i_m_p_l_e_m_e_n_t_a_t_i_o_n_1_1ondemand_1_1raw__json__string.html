Detailed Description

A string escaped per JSON rules, terminated with quote (").

They are used to represent unescaped keys inside JSON documents.

(In other words, a pointer to the beginning of a string, just after the start quote, inside a JSON file.)

This class is deliberately simplistic and has little functionality. You can compare two raw_json_string instances, or compare a raw_json_string with a string_view, but that is pretty much all you can do.

They originate typically from field instance which in turn represent key-value pairs from object instances. From a field instance, you get the raw_json_string instance by calling key(). You can, if you want a more usable string_view instance, call the unescaped_key() method on the field instance.
